HTML e CSS

Introduzione

In informatica l'HyperText Markup Language (traduzione letterale: linguaggio a marcatori per ipertesti) è un linguaggio di markup. Nato per la formattazzione e impaginazione di documenti ipertestuali, oggi è utilizzato principalmente per il disaccoppiamento della struttura logica di una pagina web e la sua rappresentazione, gestita tramite gli stili CSS(acronimo di Cascading Style Sheets, in italiano fogli di stile a cascata). Il CSS, in informatica, è un linguaggio usato per definire la formattazione di documenti HTML ad esempio i siti web e relative pagine web. L'introduzione del CSS è divenuta necessaria per programmare in modo più chiaro e facile da capire.

Come creare un file

Per iniziare basta pochissimo:
-Un browser a vostra scelta.
-Un editor di testi
anche di questi ce ne sono molti un esempio e' atom Lanciamo il nostro editor di testi e scriviamo un semplice testo come: “Hello World”. Dopo questo salviamo il file con estensione .html ad esempio “helloworld.html“, una volta fatto questo, cliccando sul file si aprirà automaticamente il browser predefinito di sistema che mostrerà una schermata contraddistinta da quello che avrete scritto nel vostro editor di testo
