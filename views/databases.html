Databases

Introduzione

Il database (in italiano base di dati,abbreviato anche con la sigla DB) in informatica indica un insieme di dati, omogeneo per contenuti e per formato, memorizzati in un elaboratore elettronico e interrogabili grazie al terminale utilizzando le chiavi di accesso previste. Ad oggi i database vengono utilizzati ad esempio nella memorizzazione dei dati degli utenti (email,password,username).

Linguaggi di programmazione

ci sono diversi tipi di linguaggi di programmazione che servono per utilizzare i database ad esempio: SQL: MySQL No SQL: MongoDB

MongoDB

è il più popolare database NoSQL e si allontana dalla struttura tradizionale basata su tabelle dei database relazionali in favore di documenti in stile JSON con schema dinamico, rendendo l'integrazione di dati di alcuni tipi di applicazioni più facile e veloce.
