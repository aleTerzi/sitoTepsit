PHP

Introduzione

Php(Php Hypertext Processor) e’ un linguaggio di programmazione che si occupa principalmente dello sviluppo lato server, Php infatti serve per elaborare dati compiere calcoli ecc, e vive dentro ad un server. Php nasce nel 1994 da Rasmus Lerdorf

Struttura sincrona

A differenza di node, Php ha qualche hanno ormai, ma nonostante questo Php e’ ancora uno dei linguaggi piu’ usati nel web, ed il piu’ utilizzato fra i linguaggi lato server. Php nonstante non sia cosi’ veloce come node, e’ un linguaggio molto solido e rodato capace di elaborare una grande quantita’ di informazioni a differenza di node Php esegue le istruzioni una per volta, rendendo il processo di elaborazione piuttosto lungo

Relazioni

PHP essendo soprattutto usato per lo sviluppo lato server si relazione o comunque viene usato insieme ad HTML e CSS, oltre ai linguaggi SQL per interpretare i databases

Xampp

Php necessita di lavorare su un server per questo molti in fase di sviluppo usano xampp, una piattaforma che permette di usufruire di un server e di varie altre tecnologie, tra cui Phpmyadmin che permette di usare i database. Primo programma

dopo aver correttamente configurato xampp, potremo creare il nostro primo fil e con estensione php questo codice e’ un esempio di primo programma dove daremo in output la frase “Hello world!” 

			
<?php>
   echo "Hello World!";
?>

tutti i nostri programmi in php andranno inseriti fra questa sintassi dopo aver salvato il file nella cartella htdocs, ed aver fatto partire il server Apache, dal pannello di controllo di xampp, digitate nell’url di un qualsiasi browser localhost/nomefile.php, dovreste vedere scritto a schermo proprio Hello world!

PHP Post

dopo aver creato il nostro primo programma possiamo veramente fare moltissime cose con Php, e qui di seguito vi mostrero’ una procedura per ricevere informazioni tramite HTML per poi elaborarle. Il codice qui sotto riceve dal un form HTML il nome di una persona, e quello stesso nome verra dato in output grazie a PHP

file PHP (esempio.php)

			
<?php>
  $nome = $_POST['name'];
  echo $nome;
?>

file HTML

			
<form action="esempio.php" method="post">
<label for="name">Nome</label>
<input type="text" name="name">
<br>
<input type="submit" value="invia">
</form>

per fare cio' e' importante che nell'attributo 'action' del form inseriate il nome del vostro file php collegato, nel mio caso e' appunto esempio.php

Conclusione
